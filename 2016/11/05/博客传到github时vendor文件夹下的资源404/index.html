博客传到github后vendor文件夹下的资源404

发表于 2016-11-05   |  

今天早上更新了一些小东西，当运行hexo d -g部署到github后，访问博客时发现内容全部无法显示，看控制台报错全是vendor文件下的资源404

如下图

然后去hexo的issue搜索了下问题，结果是next主题更新导致的问题。解决方案就是在root文件夹下添加.nojekyll文件
具体内容可以查看#1220

我先是在public文件夹下加上这个文件，但是因为里面什么内容都没有，所以并没有提交上去，后来在.deploy_git文件夹下也加了这个文件，github那边提交上去了，网站也可以正常访问了，耶~~
